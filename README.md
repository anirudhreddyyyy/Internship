# Sample Markdown Document

This is a *sample* markdown file to demonstrate the analyzer tool.

## Introduction

The markdown analyzer can count various elements in your markdown files. Here's what it can do:

- Count words
- Identify headings (h1-h6)
- Extract and validate links
- Find images
- Count code blocks and lists

### Features

#### Link Validation
